## 1. 화면 개요

- **화면 ID**: `AI-03`
- **화면명**: 분석 리포트 (Analysis Report)
- **URL 경로**: `/ai/analysis-report`
- **대상 사용자**: AiGLE 개발자 및 관리자
- **Figma Node ID**: `미정`
- **비즈니스 목적**: Prompt 아카이브에서 선택된 테스트 데이터들을 기반으로 정밀 통계 및 심층 분석 인사이트를 생성하고, 이를 시각화하여 제공하며 PDF 문서로 출력한다.

### 주요 사용자 흐름 (Happy Path)

1. 사용자가 Prompt 아카이브(AI-02) 화면에서 분석할 데이터들을 선택하고 [데이터 분석 시작] 버튼을 클릭한다.
2. 시스템이 전달받은 데이터를 기반으로 정확도, 비용, 오류 분포 등을 실시간 계산한다.
3. 분석 요약 카드, 인사이트 텍스트, 성능 비교 차트 및 상세 요약 테이블이 화면에 표시된다.
4. 사용자는 [PDF 리포트 출력] 버튼을 통해 분석 결과를 문서화한다.

## 2. 화면 구성

_사용자 액션이 필요한 기능만 기술합니다. 타이틀, 라벨 등 단순 표시는 Figma에서 확인합니다._

### 상단 헤더 및 컨트롤 영역

**전체 공통:**
- [PDF 리포트 출력] 버튼 클릭하여 **브라우저 인쇄 기능 호출 및 PDF 저장/출력 실행**
  - 성공: 인쇄 전용 CSS 적용(배경색 흰색, 그림자 제거, 불필요 UI 숨김) 후 인쇄 다이얼로그 노출
  - 실패: 브라우저 팝업 차단 시 "팝업 차단을 해제해 주세요." 안내 노출
  - 조건: 헤더와 '돌아가기' 버튼 등 불필요 UI는 제외하고 분석 리포트 내용만 포함 (Media Query: Print 대응)

### 요약 통계 대시보드 (Top Summary)

**전체 공통:**
- 분석 데이터 수, 정확도, 총 예상 비용, 취약 영역, 최다 빈출 오류 지표 표시
- 정확도 계산: `(결과 분류가 '정확'인 데이터 수 / 전체 데이터 수) × 100` (소수점 1자리까지)
- 비용 합산: 각 데이터 행의 원화(KRW) 환산 금액을 누적 합계하여 천 단위 콤마(,) 포함 표시
  - 조건: 아카이브 메뉴에서 반영된 원화 데이터를 그대로 사용, 리포트 진입 시 추가 환율 API 호출 없음
- 취약 영역 및 최다 빈출 오류: `과목` 및 `오류 유형` 필드를 빈도(Frequency) 기반 분석하여 자동 추출

### 심층 분석 인사이트 (Insights)

**전체 공통:**
- 룰 기반(Rule-based) 방식으로 아래 4가지 인사이트 텍스트 자동 생성 (별도 AI API 호출 없음):
  1. **취약 영역 식별**: 카테고리별 정답률 중 최저 성능 영역 추출 → "{최저 성능 영역} 영역에서 가장 낮은 정답률({해당 영역 정답률}%)을 보입니다. 이는 전체 평균 대비 {평균과의 격차}%p 낮은 수치로, 학습 데이터 보강이 필요합니다."
     - 조건: 교과가 1개만 존재할 경우 "비교 가능한 영역이 부족합니다." 표시
  2. **오류 원인 진단**: 전체 오류 데이터 중 최다 발생 오류 유형과 비중 계산 → "전체 오류의 약 {발생 비중}%가 '{최다 발생 오류}'에서 발생하고 있습니다. Vision 모델 해상도 및 전처리 점검이 권장됩니다."
     - 조건: 모든 데이터가 '정확'일 경우 "발생한 오류 없음" 또는 "-" 표시
  3. **평가 모드 격차**: '자율평가' 정답률과 '자동평가' 정답률 간의 차이 → "자동평가 대비 자율평가 모드의 정답률이 {정답률 격차}%p 더 높습니다/낮습니다."
  4. **시간 대비 효율성**: 평균 처리 시간이 가장 긴 영역 추출 → "가장 시간이 오래 걸리는 '{최대 지연 영역}'(평균 {지연 시간}초) 영역의 정답률은 {해당 영역 정답률}%입니다."
     - 조건: `latency` 값이 없는 데이터는 평균 계산 시 제외

### 영역별 정답률 차트

**전체 공통:**
- 가로 바 차트로 카테고리별(교과) 정답률 시각화
  - 조건: 영역별 정답률 = `(해당 교과 '정확' 데이터 수 / 해당 교과 전체 데이터 수) × 100`. 단일 교과 시 단일 바만 표시

### 평가 모드 비교 차트

**전체 공통:**
- 비교 차트로 자동(Auto) 및 자율(Autonomous) 모드 정답률 시각적 대비

### 모델 성능 상세 요약 테이블

**전체 공통:**
- 상세 요약 테이블로 영역별(교과) 문항수, 정답률, 최다오류, 평균 처리 시간 표시
- 정답률 수치 옆에 미니 프로그레스 바 노출
- 각 교과별 최다 발생 오류 유형 분석 노출
- 평균 처리 시간: `전체 처리 시간(Latency)의 합 / 문항 수` (소수점 2자리까지)
- 명칭 매핑: 시스템 내부 영문 Key(Code)를 사용자 화면 표시용 한글 레이블로 일괄 치환

## 3. 상태별 변화

_화면 상태에 따라 달라지는 부분입니다._

### AI-03:로딩

- **상태 설명**: 리포트 화면 진입 시 통계 계산 중인 상태
- **전환 트리거**: 화면 진입 → 현재 상태 진입

#### 리포트 영역

**전체 공통:**
- 각 섹션별 스켈레톤 UI 노출, PDF 출력 버튼 비활성화

---

### AI-03:리포트

- **상태 설명**: 아카이브에서 데이터가 정상 전달되어 리포트가 완성된 상태
- **전환 트리거**: 통계 계산 완료 → 현재 상태 진입

#### 리포트 영역

**전체 공통:**
- 통계 계산 후 즉시 리포트 섹션별 애니메이션과 함께 노출

---

### AI-03:빈데이터

- **상태 설명**: 전달된 데이터가 없는 상태
- **전환 트리거**: 데이터 없이 화면 진입(직접 URL 접근 등) → 현재 상태 진입

#### 안내 메시지

**전체 공통:**
- "분석할 데이터가 없습니다. 아카이브에서 데이터를 선택해 주세요." 안내 및 이동 유도

---

### AI-03:에러

- **상태 설명**: 데이터 파싱 또는 계산 오류 발생 상태
- **전환 트리거**: 파싱/계산 오류 → 현재 상태 진입

#### 안내 메시지

**전체 공통:**
- "리포트 생성 중 오류가 발생했습니다. 다시 시도해 주세요." 토스트 노출 및 재시도 유도
