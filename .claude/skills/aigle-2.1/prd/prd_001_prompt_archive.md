## 1. 화면 개요

- **화면 ID**: `AI-02`
- **화면명**: Prompt 아카이브 (Prompt Archive)
- **URL 경로**: `/ai/prompt-archive`
- **대상 사용자**: AiGLE 개발자 및 관리자
- **접근 권한**: 'neolab.net' 계정인 id 교사 사용자에게만 노출
- **Figma Node ID**: `미정`
- **비즈니스 목적**: Prompt Studio에서 수행된 테스트 결과들을 영구 저장하고, 저장된 데이터를 필터링/정렬하여 관리하며, 선택된 데이터들을 기반으로 정밀 분석 리포트를 생성한다.

### 주요 사용자 흐름 (Happy Path)

1. 사용자가 Prompt 아카이브 메뉴에 진입하여 저장된 테스트 이력 목록을 확인한다.
2. 과목별 필터링 기능 또는 정렬 기능을 사용하여 특정 테스트 데이터를 찾는다.
3. 목록에서 결과 분류(정확도)나 오류 유형을 직접 수정하여 데이터를 정제한다.
4. 분석이 필요한 데이터들을 체크박스로 다중 선택한다.
5. 상단의 [데이터 분석 시작] 버튼을 클릭하여 분석 리포트 화면으로 이동한다.

## 2. 화면 구성

_사용자 액션이 필요한 기능만 기술합니다. 타이틀, 라벨 등 단순 표시는 Figma에서 확인합니다._

### 상단 헤더 및 통계 영역

**전체 공통:**
- [데이터 분석 시작] 버튼 클릭하여 **선택된 데이터 기반 분석 프로세스 실행**
  - 성공: 분석 리포트(AI-03) 화면으로 이동
  - 실패: "분석할 데이터를 선택해 주세요." 알림 노출
  - 조건: 체크박스로 1건 이상 데이터가 선택된 경우에만 버튼 활성화

### 툴바 및 필터 영역

**전체 공통:**
- 과목 필터 Select에서 과목 선택하여 **목록 필터링**
  - 성공: 하단 테이블 목록이 즉시 갱신됨
  - 조건: 전체, 국어, 수학, 영어, 과학, 사회 중 선택
- 정렬 기준 Select에서 정렬 기준 선택하여 **목록 정렬 변경**
  - 성공: 애니메이션 없이 즉시 리스트 재배열
  - 조건: 날짜순, 평가모드 순, 과제별. 기본값은 날짜 기준 내림차순(최신순)
- 정렬 방향 버튼 클릭하여 **오름차순/내림차순 토글**
- 건수 표시 영역에 전체 데이터 수 및 현재 선택된 데이터 수 실시간 표시

### 데이터 아카이브 테이블 (목록 영역)

**전체 공통:**
- 헤더 체크박스 클릭하여 **전체 선택/해제**
  - 성공: 모든 로우의 체크박스 상태 변경, 상단 분석 버튼 내 숫자 갱신
- 개별 로우 체크박스 클릭하여 **개별 데이터 선택/해제**
  - 성공: 해당 로우의 배경색 변경 및 상단 분석 버튼 내 숫자 갱신
- 결과 분류(정확도) Select에서 값 변경하여 **정확도 수정**
  - 성공: 수정된 결과 분류가 즉시 데이터에 반영됨
  - 조건: 정확, 인접, 불일치 중 선택
- 오류 유형 Select에서 값 변경하여 **오류 유형 수정**
  - 성공: 수정된 오류 유형이 즉시 데이터에 반영됨
  - 조건: OCR오류, Hallucination, Logic Error 등 상세 오류 유형
- 비용 표시 영역에서 미국 달러(USD) 및 원화(KRW) 환산 금액 확인
  - 조건: 접속 시점의 환율을 실시간 반영, 동기화 요청 시 최신 환율 재조회
- 개별 로우의 [삭제] 버튼 클릭하여 **데이터 삭제**
  - 성공: "이 테스트 데이터를 삭제하시겠습니까?" 컨펌창 노출 → 확인 시 삭제 처리
  - 실패: 컨펌창에서 취소 시 변경 없음
  - 조건: 마지막 데이터 삭제 시 즉시 `AI-02:빈목록` 상태로 전환

## 3. 상태별 변화

_화면 상태에 따라 달라지는 부분입니다._

### AI-02:로딩

- **상태 설명**: 화면 진입 시 데이터 조회 중인 상태
- **전환 트리거**: 화면 진입 → 현재 상태 진입

#### 목록 영역

**전체 공통:**
- 테이블 영역 스켈레톤 UI 노출, 툴바 및 버튼 비활성화

---

### AI-02:목록

- **상태 설명**: 데이터 로드 완료 후 아카이브 리스트가 정상 출력된 상태
- **전환 트리거**: 데이터 로드 완료 → 현재 상태 진입

#### 목록 영역

**전체 공통:**
- 저장된 아카이브 리스트를 테이블 형식으로 출력
- 데이터가 많아질 경우 테이블 영역에 스크롤바 생성, 헤더는 고정(Sticky) 처리

---

### AI-02:빈목록

- **상태 설명**: 저장된 데이터가 하나도 없는 상태
- **전환 트리거**: 데이터 없음 또는 마지막 데이터 삭제 → 현재 상태 진입

#### 안내 메시지

**전체 공통:**
- 센터 정렬된 안내 아이콘과 "아카이브된 데이터가 없습니다. 'Prompt Studio'에서 결과를 저장해 주세요." 문구 노출

---

### AI-02:에러

- **상태 설명**: 데이터 조회 실패 상태
- **전환 트리거**: 네트워크 에러 또는 API 실패 → 현재 상태 진입

#### 안내 메시지

**전체 공통:**
- "데이터를 불러올 수 없습니다." 토스트 메시지 노출 및 재시도 버튼 제공

## 4. 관련 화면

- **`AI-01`**: Prompt Studio - 테스트를 수행하고 결과를 아카이브로 전송(저장)하는 원천 화면
- **`AI-03`**: Analysis Report - 아카이브에서 선택된 데이터들을 시각화하여 통계 리포트를 보여주는 화면
