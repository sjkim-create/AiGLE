# [GRD-BULK] 일괄 채점 화면 개발 명세서

## 1. 화면 개요

- **화면 ID**: GRD-BULK (임의)
- **화면명**: 일괄 채점
- **메뉴 경로**: 과제 및 채점관리 > 채점 관리
- **URL 경로**: `/setting` (Setting.jsx 내 일괄 채점 모달)
- **대상 사용자**: 교사
- **비즈니스 목적**: 여러 학생의 과제 데이터(스마트펜 필기 데이터 및 OCR 스캔 데이터)를 한 번에 검토하고 일괄적으로 AI 채점을 요청하여, 교사의 채점 업무 효율을 극대화하기 위함.

---

## 2. 화면 구성

_사용자 액션이 필요한 기능만 기술합니다. 타이틀, 라벨 등 단순 표시는 Figma에서 확인합니다._

### 크래들 및 펜 상태 영역

**교사 전용:**
- **크래들 이용 가이드 클릭**하여 상세 가이드 노출/숨김
  - 성공: 크래들 가이드 내용 아코디언 토글
- **크래들 연결 새로고침 클릭**하여 연결된 크래들 및 펜 상태 최신화
  - 성공: 하단 펜 목록 데이터 새로고침 및 최신 상태로 갱신

### OCR 데이터 영역 (스캔/이미지 제출)

**교사 전용:**
- **OCR 데이터 목록 및 상태 확인** 
  - 상태별 시각적 분리: 다크 네이비 테마 카드 뷰(Card View) 형태로 제공
  - 표시 정보: 학생 정보(이름, 학번), 제출 상태(미채점-제출완료 등), 실시간 채점 진행률 프로그래스 바 (AI 채점 중일 때)
- (별도 조작 없이 하단의 '일괄 채점 시작' 액션과 동시에 처리됨)

### 펜 펌웨어 업데이트 영역

**교사 전용:**
- **펌웨어 일괄 업데이트 클릭**하여 업데이트가 필요한 펜들의 펌웨어 업데이트 일괄 진행
  - 성공: 각 펜의 프로그래스 바(%) 진행 후 `최신` 버전으로 상태 변경됨
  - 조건: 연결된 펜 중 `업데이트 필요` 상태인 펜이 존재할 때 활성화 및 이용 가능

### 일괄 채점 제어 영역

**교사 전용:**
- **일괄 채점 시작 클릭**하여 화면에 표시된 대상(펜, OCR)에 대한 AI 채점 요청을 서버로 전송
  - 성공: 리스트 각 항목의 상태가 `AI 채점중`으로 변경되고 진행률 프로그래스 바 표출, 상단 통계(채점 중/완료) 수치 업데이트
  - 실패: 처리 오류 발생 항목은 `채점 실패` 정보 표출 (붉은 배지)
  - 조건: 업데이트가 필요한 펜이 없고 펌웨어 업데이트가 진행 중이 아닐 때 버튼 활성화
- **닫기 (모달 우측 상단 X 아이콘 또는 하단 버튼) 클릭**하여 일괄 채점 모달 닫기
  - 성공: 채점 대기 상태 및 완료 상태일 때 모달 즉시 종료 후 채점 관리 메인 목록으로 복귀
  - 분기: 채점이 진행 중일 때 클릭 시 채점 중단 경고 모달(GRD-BULK:CONFIRM_CLOSE) 표시 (백그라운드 채점 안내)

## 3. 상태별 변화

_화면 상태에 따라 달라지는 부분입니다._

### GRD-BULK:CHECKING

- **상태 설명**: 일괄 채점 시작 전, 로컬 PC의 NeoStudio2Lite 프로그램 구동/설치 여부를 확인하는 상태
- **전환 트리거**: 메인 화면에서 `일괄 채점` 버튼 클릭 시 진입 (자동 진행) → 확인 결과에 따라 `GRD-BULK:INSTRUCTION` 또는 `GRD-BULK:NOT_INSTALLED` 로 전환

### GRD-BULK:NOT_INSTALLED

- **상태 설명**: NeoStudio2Lite가 설치되어 있지 않아 펜 데이터 동기화를 진행할 수 없는 상태
- **전환 트리거**: 화면 내 `수동 설치 안내` 버튼 클릭 시 → 설치 가이드를 포함한 메뉴얼 노출 화면(MANUAL_INSTALL)으로 전환

### GRD-BULK:INSTRUCTION

- **상태 설명**: 일괄 채점을 시작하기 전 인원 안내 통계, 브라우저 유지 등의 안내 텍스트 노출
- **전환 트리거**: `채점 시작하기` 버튼 클릭 시 → `GRD-BULK:READY` 로 전환

### GRD-BULK:READY

- **상태 설명**: 펜 목록 및 상태, OCR 데이터 목록이 성공적으로 로드되어 사용자의 채점 결정을 대기 중인 상태
- **전환 트리거**: 하단 `일괄 채점 시작` 버튼 클릭 시 → `GRD-BULK:PROCESSING` 로 전환

#### 펌웨어 업데이트 기능
**교사 전용:**
- 펌웨어가 다소 낮은 펜이 있다면 `일괄 채점 시작` 버튼이 비활성화. 이때 하단 `펌웨어 일괄 업데이트` 버튼을 누르면 업데이트가 먼저 진행된 후 일괄 채점 기능이 활성화됨.

#### OCR 데이터 연동
**교사 전용:**
- 펜 연결 목록과 분리된 다크 테마 영역에서 OCR 대상 학생이 있는지 확인 가능하며, 일괄 채점 진행 시 해당 목록도 동일하게 상태(대기 -> 채점 중 -> 채점 완료)가 업데이트됨.

### GRD-BULK:PROCESSING

- **상태 설명**: 여러 개의 답안 데이터에 대한 AI 채점이 병렬/순차적으로 진행 중인 상태 (리스트 내 각 항목별 프로그래스 연출)
- **전환 트리거**: 모든 리스트의 채점이 100% 성공 혹은 실패로 처리 완료 시 → `GRD-BULK:COMPLETED` 로 전환
- **전환 트리거**: 채점 중단 목적으로 우측 하단 `닫기` 클릭 시 → 종료 동의를 구하는 팝업 노출 후 승인 시 백그라운드로 전환 (모달은 닫히고 메인 리스트 우측 하단에 FAB 형태로 상태 지속 연출)

### GRD-BULK:COMPLETED

- **상태 설명**: 펜 및 OCR의 모든 일괄 채점이 완전히 종료된 상태
- **전환 트리거**: 상단에 채점 완료를 알리는 녹색 가이드 배너 표출. 사용자가 `닫기` 클릭 시 → 모달 닫힘

## 4. 관련 화면

- **GRD-01 (채점 관리 본 화면)**: 채점 관리 - 목록에서 대상 학생을 선택하고 우측 하단에 뜨는 일괄 채점을 진입하는 메인 화면
- **SET-01 (환경 설정 화면)**: 환경 설정 - 일괄 펌웨어 업데이트 등 기기가 연결된 전반적인 시스템 상태를 확인 및 조작하는 기능 연계 화면
