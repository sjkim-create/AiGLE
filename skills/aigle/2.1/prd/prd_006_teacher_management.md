# [TCH-01] 교사 관리 화면 개발 명세서

## 1. 화면 개요

- **화면 ID**: `TCH-01`
- **화면명**: 교사 관리
- **메뉴 경로**: `회원 > 교사 관리`
- **URL 경로**: `/?menu=member&subMenu=teacher-management`
- **대상 사용자**: 학교 관리자(학교 모드), 시스템 관리자(시스템 모드)
- **비즈니스 목적**: 각 학교에 소속된 교사 정보를 관리하고, 시스템 차원에서 전체 교사 회원의 현황을 파악 및 상태를 관리하기 위함.

---

## 2. 화면 구성

_사용자 액션이 필요한 기능만 기술합니다. 타이틀, 라벨 등 단순 표시는 Figma에서 확인합니다._

### 검색 및 필터 영역

**학교 모드 (학교 관리자) 전용:**
- 교사명 검색어 입력 후 아이콘 클릭 또는 엔터하여 교사 목록 필터링
  - 성공: 검색 조건에 맞는 교사 목록 노출
  - 실패: "아직 등록된 교사가 없습니다." 메시지 노출

**시스템 모드 (시스템 관리자) 전용:**
- 드롭다운 또는 검색을 통해 학교 선택하여 해당 학교 기준으로 목록 필터링
- 교사 이름 또는 아이디 검색어 입력하여 통합 검색
- 회원 유형 체크박스(전체 / 학교 유료 회원 / 무료 회원) 선택하여 단일 선택 필터링
  - 조건: 기본값은 '전체' checked

---

### 목록 및 기능 영역

**전체 공통:**
- [+ 교사등록] 버튼 클릭하여 교사 등록 화면(`TCH-02`)으로 이동
- 체크박스 클릭하여 교사 항목 개별 선택 또는 전체 선택
- 페이징 인터페이스 클릭하여 목록 페이지 이동

**학교 모드 (학교 관리자) 전용:**
- 교사 항목 클릭하여 소속 교사 상세 정보 확인 및 수정

**시스템 모드 (시스템 관리자) 전용:**
- 학교 변경 신청 / 학교 유료 회원 승인 신청 상태의 교사에 대해 관리 액션 처리
  - 조건: 상태 열에 붉은색 텍스트로 '학교 변경 신청' 또는 '학교유료회원 승인신청' 노출 시 활성화

## 3. 상태별 변화

### TCH-01:목록

- **상태 설명**: 화면 진입 시 필터링이 적용되지 않은 전체 교사 목록 노출 (초기 상태)
- **전환 트리거**: 검색어 입력 또는 필터 선택 → `TCH-01:검색결과`

#### 목록

**전체 공통:**
- 전체 교사 목록이 기본 정렬 기준으로 노출됨

**시스템 모드 (시스템 관리자) 전용:**
- 학교 변경 신청 상태 교사: 상태 열에 붉은색 텍스트로 '학교 변경 신청' 및 관련 학교명 병기
- 학교유료회원 승인신청 상태 교사: 상태 열에 '학교유료회원 승인신청' 텍스트 노출

---

### TCH-01:검색결과

- **상태 설명**: 검색어 또는 필터 조건에 맞는 데이터만 목록에 바인딩된 상태
- **전환 트리거**:
  - 검색어/필터 초기화 → `TCH-01:목록`
  - 결과 없을 경우 → `TCH-01:빈결과`

#### 검색 결과 목록

**전체 공통:**
- 조건에 맞는 교사 목록 노출

---

### TCH-01:빈결과

- **상태 설명**: 검색 조건에 해당하는 교사가 없는 상태
- **전환 트리거**: 검색어/필터 변경 → `TCH-01:검색결과` 또는 `TCH-01:목록`

#### 안내 메시지

**전체 공통:**
- "검색 결과가 없습니다" 안내 메시지 노출

---

## 4. 관련 화면
